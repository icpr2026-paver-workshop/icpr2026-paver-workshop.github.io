# ICPR 2026 Workshop Website

A professional website for the ICPR 2026 workshop, built with HTML, CSS, and JavaScript.

## Features

- Modern responsive design
- Gradient hero section
- Multiple content sections (About, Speakers, Schedule, Call for Papers, Organizers, Contact)
- Mobile-friendly navigation
- Smooth scrolling

## Local Development

Simply open `index.html` in your web browser to view the website locally.

## Hosting

This website can be hosted on:
- GitHub Pages (free)
- Netlify (free)
- Vercel (free)
- Any web hosting service

## Customization

Replace all placeholder text in brackets `[...]` with your actual workshop information:
- Workshop title and subtitle
- Speaker names and details
- Important dates
- Organizer information
- Contact details

## Structure

- `index.html` - Main HTML file
- `styles.css` - Stylesheet
- `script.js` - JavaScript for interactive features
